# `component()`

A custom component for a navigation item,
for if you want to add custom functionality to your sidebars (or other navigation structures).


## Example

```tsx title="theme.config.tsx"
function DonateButton() {
  return (
    <div>
      <Button>Donate</Button>
    </div>
  )
}

export default defineTheme({
  directories: [
    directory("dir-a", {
      sidebar: [
        link("Home", "/home"),
        component(() => <DonateButton />)
      ]
    })
  ],
})
```
